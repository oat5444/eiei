# User Stories

## User Story 1

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ ระบุจำนวนความยาวของรหัสผ่านได้\]\
**So that** \[เพื่อให้ รหัสผ่านที่ได้ตรงตามข้อกำหนดของเว็บไซต์ที่จะนำไปใช้\]

------------------------------------------------------------------------

## User Story 2

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ ให้ผสมตัวอักษรพิมพ์เล็กพิมพ์ใหญ่ได้\]\
**So that** \[เพื่อให้ สามารถกำหนดความซับซ้อนของรหัสผ่านได้\]

------------------------------------------------------------------------

## User Story 3

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ ให้ระบบตั้งค่าเริ่มต้นมาให้เลย (เช่น ความยาว 12 ตัวอักษร และเลือกทุกเงื่อนไข)\]\
**So that** \[เพื่อให้ ฉันสามารถกดปุ่ม Generate ได้โดยไม่ต้องเสียเวลากำหนดค่าเองทุกครั้ง\]

------------------------------------------------------------------------

## User Story 4

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ ปุ่มสำหรับคัดลอกรหัสผ่าน\]\
**So that** \[เพื่อให้ ฉันนำรหัสผ่านไปวาง ใช้งานได้ทันทีโดยไม่ต้องเสียเวลาพิมพ์ตามหรือลากคลุมข้อความ\]

-------------------------------------------------------------------------
## User Story 5

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ กดปุ่ม Generate ซ้ำได้เรื่อยๆ โดยไม่ต้องรีเฟรชหน้าจอ\]\
**So that** \[เพื่อให้ ฉเลือกดูรหัสผ่านหลายๆ รูปแบบจนกว่าจะเจอชุดที่พอใจ\]

-------------------------------------------------------------------------

## User Story 6

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ เห็นรหัสผ่านที่สร้างขึ้นด้วยฟอนต์ที่อ่านง่าย\]\
**So that** \[เพื่อให้ ฉันสามารถแยกแยะความแตกต่างระหว่างตัวอักษรที่คล้ายกันได้ถูกต้อง (เช่น ตัว L เล็ก , ตัว i ใหญ่ , และเลข 1)\]

-------------------------------------------------------------------------

## User Story 7

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ เห็นข้อความแจ้งเตือนเมื่อฉันลืมเลือกประเภทตัวอักษร (ไม่เลือกเลยสักช่อง)\]\
**So that** \[เพื่อให้ ฉันรู้ตัวและกลับไปเลือกเงื่อนไขก่อนกดสร้างรหัสผ่านอีกครั้ง\]

-------------------------------------------------------------------------

## User Story 8

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ ให้ระบบทำงานได้รวดเร็วและรองรับการใช้งานบนเบราว์เซอร์\]\
**So that** \[เพื่อให้ ฉันสามารถเข้ามาสร้างรหัสผ่านได้ทันทีโดยไม่ต้องติดตั้งโปรแกรมลงในเครื่อง\]

-------------------------------------------------------------------------

## User Story 9

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ ปรับความยาวรหัสผ่านได้ง่ายๆ (เช่น พิมพ์ตัวเลข หรือ เลื่อน Slider)\]\
**So that** \[เพื่อให้ ฉันได้รหัสผ่านที่มีความยาวตามที่ต้องการ\]

-------------------------------------------------------------------------
## User Story 10

**As a** \[ในฐานะ ผู้ใช้\]\
**I want to** \[ฉันต้องการ เลือกผสมตัวอักษร ตัวเลข และสัญลักษณ์ได้อย่างอิสระ\]\
**So that** \[เพื่อให้ ฉันสามารถสร้างรหัสผ่านที่ซับซ้อนตามที่ต้องการได้\]

-------------------------------------------------------------------------
